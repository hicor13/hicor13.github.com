¡Bienvenidos a mariocornejo.com!

Esta es una página web que he creado para concentrar todos mis esfuerzos creativos, siempre he querido tener una página web y bueno ya estoy aquí. No es mi primera página web, pero es la primera que he construido completamente usando HTML, CSS y JavaScript. Así que la página web solo mejorará con el pasar del tiempo.

mariocornejo.com es una página web construida con la intención de exponer los conocimientos que he ido adquiriendo a lo largo del tiempo. Es un proyecto mediante el cuál busco mejorar mis habilidades y, en el camino crear una página web bien interesante 😎.
